# Simple Recipe Webpage (HTML Only)

This project is a static webpage designed to display multiple recipes with their ingredients and instructions. It is built using **pure HTML5** to demonstrate basic content structuring and semantic HTML elements without relying on CSS for styling or JavaScript for interactivity.

* **Recipe Organization:** Each recipe is contained within a `<section>` for clear separation.
* **Structured Data:** Uses semantic lists (`<ul>` for ingredients, `<ol>` for instructions) to present information clearly.
* **Image Placeholders:** Includes `<figure>` and `<img>` tags for displaying recipe visuals (placeholders are used, as actual image files are not included).
* **Clear Hierarchy:** Utilizes `<h1>` and `<h2>` tags to establish a logical content hierarchy.
* **Basic Metadata:** Includes essential elements like `<head>` with `title` and `meta` tags.

## ‚ö†Ô∏è Limitations (HTML Only)

Since this project strictly adheres to the "HTML Only" constraint, it has the following limitations:

* **No Styling:** The page appearance is the browser's default‚Äîit lacks custom fonts, colors, borders, or sophisticated layout, as these require **CSS**.
